== Description ==

CommentRef was inspired by [CommentLuv](https://wordpress.org/plugins/commentluv/) after it leaked of update.

The plugin provides awesome features that help with building bigger blogging community, engage more reading, and generating more leads to your blog. CommentRef adds a link to a recent post of the commentators to encourage more reading and commenting. It is brillant ways to provides more useful resources on your blog.

Major features in CommentRefs includes:

*   Add a link of commentator's recent post underneath the comment content
*   Share to get 10 post choice
*   Configure dofollow or nofollow
*   First comment redirect
*   Low quality comment filtering
*   CommentLuv migration
